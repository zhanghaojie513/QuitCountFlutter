# 最新 Web 原型同步清单

本清单用于把 `quit-smoking-prototype-preview` 当前代码和截图同步到 Flutter 移动端实现。实现时以 `docs/product_spec.md` 为产品规则源，以最新 Web 原型为 UI/流程基准。

## 首页

- [ ] 首页不显示“查看资产”按钮。
- [ ] 首页操作区只保留“放回库存”和“取出一支”。
- [ ] “放回库存”撤销一次真实成本，今日取出为 0 时置灰。
- [ ] 主卡信息使用 `库存 17 支 · 今日 4/5 支` 这类紧凑文案。
- [ ] “当日危害说明”小 i 位于“今日危害评分”标题旁。
- [ ] 首页不显示重复的“当日危害：高危”警示条。
- [ ] 首页顺序为：主卡 -> 三指标 -> 7 日危害趋势 -> 操作按钮 -> 最近账本。

## 趋势图

- [ ] 保留 7 日危害趋势。
- [ ] 趋势图视觉保持紧凑。
- [ ] 去掉重复“持平”小文案，只保留标题和右侧状态，如“稳定”。

## 每日复盘

- [ ] 每日复盘弹窗改为图表优先。
- [ ] 核心图表为“今日 vs 目标线”柱状图。
- [ ] 下方保留真实日耗、焦油摄入、健康折算三项指标。
- [ ] 不照搬旧版大段说明。

## 取烟与严厉提醒

- [ ] 取烟 Sheet 中备注常驻显示。
- [ ] 备注不是必填。
- [ ] 记录取出后直接扣库存、写账本、更新首页。
- [ ] 严厉危害警示不阻止取烟。
- [ ] 达到干预线后只触发手机系统通知栏三连提醒，APP 内不做三连提示。

## 验证基准

- [ ] 首页对照 `qa\screenshots\home-actions-simplified.png`。
- [ ] 页面验收继续覆盖 `390x844` 核心视口，并保持响应式布局，不写死页面尺寸。
